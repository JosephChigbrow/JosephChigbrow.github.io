Below is some progress I have made on my Individual Game for the Boise State GIMM program. It's a survival game placed in a small suburban town where players must creatively problem solve living as long as possible while keeping a quality reputation with the townsfolk.

As seen above, I have also began work towards implementing Unity VFX into my projects as well. This particular effect will be flipped on an axis and displayed face down on the ground to act almost as a glowing, reverberating waypoint. This could be really helpful for the first few play throughs if players need extra help knowing how to fully utilize the town.
