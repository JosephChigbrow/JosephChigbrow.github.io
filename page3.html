Below is some progress I have made on my Individual Game for the Boise State GIMM program. It's a survival game placed in a small suburban town where players must creatively problem solve living as long as possible while keeping a quality reputation with the townsfolk.

As seen above, I have also began work towards implementing Unity VFX into my projects as well. This particular effect will be flipped on an axis and displayed face down on the ground to act almost as a glowing, reverberating waypoint. This could be really helpful for the first few play throughs if players need extra help knowing how to fully utilize the town.

This is inventory system I was working on with a team as part of my major. My job was to make a togglable inventory system that allows players to work on different recipes while exploring the world. This disables player movement at the same time and allows the player cursor to move around in the inventory rather than moving the camera and causing confusion.

These are the very first projects I worked on learning 3D modeling with Blender. These models would be used in my Individual Game as seen above. The lamp post and building are already where they need to be for awhile, but I plan on further rigging the retirever model so that it can chase the player around in game.
